Washes 2.x

The API 2.0 surface, live: one way per concept — normalized coordinates, one run policy, chainable everything. Every control is labelled with the call it makes.

Tool — pointer on the canvas
paint() dabs; stroke() lays a continuous line (pen lifts on release — penUp()); stir() pushes the fluid; lift() re-suspends dried pigment; blot() dabs it away. All take normalized 0..1 coords, radii as fractions of the smaller side.
Brush — pigment · brushSize(fraction)
Size — a fraction of the smaller side, so it means the same thing at every resolution
brushSize(0.04)
Run state — run(policy) · drying(v)
sim idle — live via the typed events on('active'|'idle'|'dry')
Determinism — create({ seed })
Rebuilds the instance with this seed and replays the opening script — the splashes land in the same seeded-random places every time. Change the seed for a different painting; keep it for the identical one.
Painting as data — saveState · loadState · exportImage
Snapshot the painting itself (fluid, pigment, deposit, paper — bit-exact), wreck the canvas, restore.
Cells & water — grid.* · splash · mask/unmask
The mosaic thinks in cells (wc.grid.size() + wc.grid.paint() — the explicit cell-space home). Splash takes normalized epicenters now. The window freezes everything except a rounded cut-out, then floods it.
Ambient — backgroundAnimation · animation
Everything above chains — wc.paperColor('#…').paintLoad(x).run('auto') is one expression in v2.
