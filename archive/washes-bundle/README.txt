Washes — bundle contents
========================

Current shipping artifacts (v0.98)
----------------------------------
  washes-v0.98.html           Full playground with all features and UI
  washes-pkg-v0.98.tar.gz     npm package tarball (src + types + examples)
  watercolor-lib.js           Standalone library file (drop into any page)
  washes.d.ts                 TypeScript definitions

Demos
-----
  washes-texture-demo.html       Brush textures (Crayon resist, Dry brush,
                                 Salt, Splatter) — v0.98's headline feature
  washes-portfolio-demo.html     Three-card portfolio with hover ripples,
                                 ink text, click-to-obliterate
  washes-threejs-demo.html       Paint on left → 3D textured mesh on right
  watercolor-thumbnails.html     Standalone 2D-canvas thumbnail filter
                                 (K-M math, no WebGL dependency)
  multi-watercolor-v0.4.html     Multiple instances on one page

Examples

  watercolor-simple-example.html  Minimal getting-started page
  watercolor-multi-example.html   Multiple instances
  watercolor-lib-test.html        Smoke test for lib integration
